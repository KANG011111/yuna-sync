## 🚀 GitHub Pages 部署步驟

### 1. 初始化 Git 倉庫
```bash
git init
git add .
git commit -m "🎉 HYPER SYNC 虛擬偶像銳舞派對網站完成"
```

### 2. 推送到 GitHub
```bash
git remote add origin https://github.com/你的用戶名/EDMCONCERT.git
git branch -M main
git push -u origin main
```

### 3. 啟用 GitHub Pages
1. 倉庫 Settings → Pages
2. Source: Deploy from a branch
3. Branch: main
4. Save

### 4. 你的網站網址
`https://你的用戶名.github.io/EDMCONCERT/`
